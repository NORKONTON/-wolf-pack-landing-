# Wolf Pack Landing Page

One-page website for Wolf Pack - play-to-earn Telegram game on TON blockchain.

## Features

- Responsive design (mobile-friendly)
- Modern dark theme with orange accents
- Sections: Hero, About Game, Tokenomics, Roadmap, Team, Play Now
- Interactive timeline for roadmap
- Bootstrap 5 + Font Awesome icons
- Smooth scrolling animations
- Ready for deployment

## Deployment Instructions

Choose one of the following free hosting options:

### Option 1: Netlify (Recommended)

1. Go to [netlify.com](https://netlify.com) and sign up (free)
2. Drag and drop the `wolf-pack-landing` folder to Netlify drop zone
3. Your site will be deployed to `random-name.netlify.app`
4. You can customize domain in Netlify settings

**Or using Netlify CLI:**
```bash
npm install -g netlify-cli
netlify deploy
```

### Option 2: GitHub Pages

1. Create a new GitHub repository: `wolf-pack-landing`
2. Upload all files to the repository
3. Go to Settings → Pages → Source: select `main` branch
4. Your site will be available at `username.github.io/wolf-pack-landing`

### Option 3: Vercel

1. Go to [vercel.com](https://vercel.com) and sign up
2. Import your Git repository or drag & drop folder
3. Vercel automatically deploys and gives you `*.vercel.app` domain

### Option 4: Cloudflare Pages

1. Sign up at [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect Git repository or upload files
3. Deploy with Cloudflare's global CDN

## Customization

1. **Change colors**: Edit CSS variables in `style.css` (`:root` section)
2. **Update content**: Edit `index.html` sections
3. **Replace favicon**: Replace `favicon.svg` with your own logo
4. **Add analytics**: Add Google Analytics or other tracking in `index.html` `<head>`
5. **Change Telegram link**: Update Telegram channel link in multiple places
